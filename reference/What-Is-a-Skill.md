# What Is a Skill?

A skill is a reusable, documented unit of assistant work.

## Minimum Skill Qualities
- clear purpose
- explicit input contract
- explicit output contract
- decision rules
- escalation boundaries
- examples and tests

## What a Skill Is Not
- a vague one-line prompt
- undocumented trial-and-error behavior
- hidden business logic that cannot be reviewed

## Why Skills Matter
Skills make assistant behavior teachable, testable, and portable across tools.
